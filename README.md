# ✨ 简介

**妙笔** —— 基于大小模型的在线文档富文本编辑器

> 2024年中国软件杯A10赛题

后端仓库请移步[SmartEditor_backend](https://github.com/electronic-pig/SmartEditor_backend)

# 🎉 特性

## 整体功能

- 用户认证
- 文档管理
- 富文本编辑
- AI功能
  
## 赛题要求

- 智能润色
- 多媒体信息提取
- 智能格式排版

# 🚀 运行

在执行后续步骤之前，请确保您配置好了以下环境：

- nodejs ≥ 18.19.1

然后在项目根目录执行以下步骤：

### 安装依赖
```sh
npm install
```

### 编译运行

```sh
npm run dev
```

### 打包部署

```sh
npm run build
```
